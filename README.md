# 📱 PokeWiki App

RxSwift와 UIKit을 사용해 만든 포켓몬 도감 앱입니다.  
PokeAPI에서 데이터를 불러와 포켓몬 리스트를 컬렉션 뷰로 보여주고, 선택한 포켓몬의 상세 정보를 확인할 수 있습니다.

---

## ✅ 주요 기능

- 포켓몬 리스트 조회
- 포켓몬 이미지 표시
- 포켓몬 클릭 시 상세 정보 화면으로 이동
- 상세 정보: 이름, 타입, 키, 몸무게 표시

---
